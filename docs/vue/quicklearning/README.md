# 快速上手

::: explain 第一步 安装组件库

```vue explain
yarn add zx-pc-design
```

:::

---

::: explain 第二步 在 main.js || main.ts 下引入

```js explain
import zui from "zx-pc-design";
import "zx-pc-design/dist/z-style.css";
```

:::
